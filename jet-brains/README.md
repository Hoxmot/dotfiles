## IntelliJ

### Intallij - Settings

* Keymap:
  * Git annotate -> `Ctrl + K`
  * Navigate back -> `Ctrl + NumPad-4`
  * Navigate forward -> `Ctrl + NumPad-6`
* Editor
  * Color Scheme -> _One Dark_
    * Language Defaults
      * Semantic Highlight -> `TRUE`
  * Font:
    * Font: _JetBrains Mono_
    * Size: `13`
    * Line height: `1.2`
    * Enable ligatures: `TRUE`
